# AppImage packaging instructions (guide)

# 1. Create an AppDir with your binary and dependencies
# 2. Use appimagetool to produce an AppImage
# Example:
# mkdir -p AppDir/usr/bin
# cp dist/StemwerkApp AppDir/usr/bin/stemwerk
# cp -r installers AppDir/usr/share/stemwerk/installers
# wget https://github.com/AppImage/AppImageKit/releases/download/continuous/appimagetool-x86_64.AppImage
# chmod +x appimagetool-x86_64.AppImage
# ./appimagetool-x86_64.AppImage AppDir

# The AppImage can include the installers folder and run the bundled installer helper.
